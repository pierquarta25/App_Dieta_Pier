# 🥗 Piano Nutrizionale — Pier

Dashboard interattiva mobile-first per la gestione del piano alimentare giornaliero.  
Ottimizzata per essere usata in palestra o in cucina, direttamente dal telefono.

---

## 🚀 Live

> 🔗 **[dieta-pier.vercel.app](https://dieta-pier-v8r2.vercel.app/)** 

---

## ✨ Funzionalità

- 🔥 **Toggle Giorni ON / OFF** — switch animato con macro aggiornati in tempo reale
- 📊 **Macro Hero** — calorie, proteine, carboidrati e grassi sempre visibili
- 🃏 **Meal Cards espandibili** — tap sulla card per vedere tutte le opzioni di ogni pasto
- 💊 **Sezione Supplementi** — stack giornaliero completo con dosaggi
- 📋 **Note del Nutrizionista** — acqua, sale, dolcificanti, pasto libero
- 📱 **Mobile-First** — progettata per essere usata da telefono

## 📲 Aggiungi a schermata Home

Per usarla come un'app:

**iPhone (Safari)**
1. Apri il link Vercel in Safari
2. Tocca l'icona **Condividi** ↑
3. Scorri e tocca **"Aggiungi a schermata Home"**
4. Conferma → appare l'icona sulla home

**Android (Chrome)**
1. Apri il link Vercel in Chrome
2. Tocca i **tre puntini** in alto a destra
3. Tocca **"Aggiungi a schermata Home"**
4. Conferma → appare l'icona sulla home
